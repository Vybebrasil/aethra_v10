# Avisos de conteúdo de terceiros

O catálogo de criaturas deste protótipo foi derivado do arquivo `pocketdm_monstros.json`, cuja metadata informa:

- Fonte: `https://pocketdm.com.br/monstros`
- Extração: 2026-07-20
- Conteúdo SRD sob CC-BY-4.0
- Conteúdo homebrew pertencente aos respectivos autores

Este pacote inclui somente os registros marcados como `is_srd: true`. Os registros homebrew da fonte original não foram distribuídos no catálogo executável.

As URLs de tokens são preservadas apenas como metadata de origem (`sourceTokenUrl`). O runtime não utiliza essas imagens automaticamente; sprites próprias ou devidamente licenciadas devem ser adicionadas em `assets/entities/`.
